Soal 1 – Konflik Nama Antar Extension

Pada kode tersebut terjadi konflik karena terdapat dua extension dengan nama method yang sama yaitu toInt() pada tipe data String.

Walaupun tipe return berbeda, Dart tetap menganggap pemanggilan method tersebut ambigu sehingga program tidak dapat dikompilasi.

Solusi yang dapat dilakukan yaitu mengganti nama method atau menggunakan extension override agar kedua extension tetap dapat digunakan tanpa konflik.

DART SOURCE CODE
// Nama  : Uliya Gumpari
// NIM   : 251410008
// Kelas : SI2A

extension StringNumber on String {

  int convertToInt() {

    return int.parse(this);
  }
}

extension StringDouble on String {

  double convertToDouble() {

    return double.parse(this);
  }
}

void main() {

  String angka = '123';

  print(angka.convertToInt());

  print(angka.convertToDouble());
}

Soal 2 – Extension Method dan Member Private

Kode gagal dikompilasi karena extension tidak dapat mengakses member private yang ditandai dengan simbol underscore (_).

Member private hanya bisa digunakan di dalam class yang sama sehingga extension di luar class tidak memiliki akses langsung ke _value.

Cara memperbaikinya yaitu dengan menambahkan getter publik pada class tanpa mengubah _value menjadi public.

DART SOURCE CODE
// Nama  : Uliya Gumpari
// NIM   : 251410008
// Kelas : SI2A

class Counter {

  int _value = 0;

  void increment() {

    _value++;
  }

  int get value {

    return _value;
  }
}

extension CounterStats on Counter {

  int getValue() {

    return value;
  }
}

void main() {

  var counter = Counter();

  counter.increment();

  print(counter.getValue());
}
